What is an exogenous variable in time series modeling

  • Endogenous: Input variables that are influenced by other variables in the system and on which the output variable depends.
  • Exogenous: Input variables that are not influenced by other variables in the system and on which the output variable depends.

What are the components of time series?

Additive decomposition: \(y_t = S_t + T_t + R_t\)

Multiplicative decomposition: \(y_t = S_t \times T_t \times R_t\)

  • \(S_t\): seasonal component
  • \(T_t\): trend component
  • \(R_t\): remainder component

ARIMA Models

  • Exponential smoothing and ARIMA are the two most widely used approaches to time series forecasting.
  • Exponential smoothing models: based on a description of trend and seasonality in the data.
  • ARIMA: aim to describe the autocorrlations in the data

Autoregressive Model

  • Autogregressive: Regression of the variable against itself.
  • Forecast the variable of interest using a linear combination of \(\textit{past values of the variable}\).
  • AR(p): \(y_t = c + \phi_1 y_{t-1} + \phi_2 y_{t-2} + ..... \phi_p y_{t-p} + \epsilon_t\)

Moving average Model

  • Moving average model uses past forecast errors in a regression-like model
  • MA(q): \(y_t = c + \epsilon_t + \theta_1 \epsilon_{t-1} + \theta_2 \epsilon_{t-2} + ..... + \theta_p \epsilon_{t-p}\)

Relation between AR and MA models

  • Any AR(p) model can be written as MA(\(\infty\)) model.
  • Example for AR(1) model
    • \(y_t = \phi_1 y_{t-1} + \epsilon_t\)
    • \(y_t = \phi_1 (\phi_1 y_{t-2} + \epsilon_{t-2}) + \epsilon_t\)
    • \(y_t = \phi_1^2 y_{t-2} + \phi_1 \epsilon_{t-2} + \epsilon_t\)
    • \(y_t = \phi_1^3 y_{t-3} + \phi_1^2 \epsilon_{t-2} + \phi_1 \epsilon_{t-1} + \epsilon_t\)
    • Eventually
    • \(y_t = \epsilon_t + \phi_1\epsilon_{t-1} + \phi_1^2\epsilon_{t-2} + \phi_1^3\epsilon_{t-3} + ...\)
    • Which is a MA(\(\infty\)) model
