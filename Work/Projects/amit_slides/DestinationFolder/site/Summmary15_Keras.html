Keras questions

What is tensor in tensorflow?

A tensor is a mathematical object represented as arrays of higher dimensions. These arrays of data with different dimensions and ranks fed as input to the neural network are called “Tensors.”

What are programming elements in Tensorflow?

  • Constants: values don’t change
    • a = tf.constant(3.0, tf.float32)
  • Variables: Variables allow us to add new trainable parameters to graph.
    • W = tf.Variable([.3].dtype=tf.float32)
  • Place holders: these allow us to feed data to a tensorflow model from outside a model.
    • a = tf.placeholder (tf.float32)
    • b = a*2
  • Sessions

Different kinds of tensors in Tensorflow?

  • tf.constant
  • tf.placeholder
  • tf.SparseTensor
  • tf.Variable
